# ejercicios-vim

Este repositorio contiene un conjunto de lecciones para el aprendizaje de vim
pensadas para usarse en un curso presencial o exposición sobre el tema. Los
archivos presentes no están pensados para ser un tutorial o una guia de
auto-aprendizaje sino para servir de temario y a la vez campo de pruebas para
una cátedra asistida.

Aunque el material en si puede resultar confuso, puede ser útil leerlo ya que
contiene hace referencia a una serie de páginas y materiales de aprendizaje
que pueden resultar útiles si estás aprendiendo vim. Si además de simplemente
leerlo, quiere hacer el intento de seguir las instrucciones usando el editor
vim será necesario que tenga un mínimo conocimiento sobre el uso del mismo.
La forma recomendable de hacerlo es la siguiente:

    * Invoque vim para abrir los documentos .txt: vim *.txt
    * Lea la lección, una vez finalizada use :next para ir a la siguiente
    * Puede abrir enlaces que se encuentre posicionando el cursor sobre ellos y
      presionando 'gx'
